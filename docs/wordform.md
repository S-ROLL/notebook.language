# <h1 style="text-align: center;">WORD FORM THEORETICAL</h1>
## Noun 🧒
    - Đếm được
        - Số ít ➡️ `` a/an + N ``.
        - Số nhiều ➡️ ``to be``.
    - Không đếm được ➡️ ``to be``.
## Verb 🏃
The same.
!!! note
    Intransitive ➡️ ``V``. Transitive ➡️ ``V + O``.
## Adj 🥀 / V3ed (passive) 💥
- ``To be`` + Adj
- Adj + ``Noun``
- ``Linking verb`` + Adj
- ``make / find / keep`` + Adj
- ``leave + O`` + Adj
